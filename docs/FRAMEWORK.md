# AI Ownership and Accountability: A Unified Policy Framework

> See the full policy document (docx) for the complete framework with 60+ citations.
> This file provides a quick reference summary.

## Five Universal Principles

1. **Consistent Attribution** — Claiming ownership and disclaiming accountability are incompatible
2. **Meaningful Human Control** — Ownership threshold = accountability threshold
3. **Proportional Accountability** — Responsibility scales with direction and foreseeability
4. **Traceability** — Provenance is prerequisite for both ownership and accountability
5. **Non-Waivable Accountability** — ToS cannot disclaim foreseeable harm liability
